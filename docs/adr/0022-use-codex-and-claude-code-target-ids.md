# Use `codex` and `claude-code` as v1 target ids

Harness v1 target ids will be `codex` and `claude-code`. The Claude target uses `claude-code` instead of `claude` because Harness installs files and configuration for the Claude Code product specifically, not Claude web, desktop, or API surfaces.

## Consequences

Target ids remain explicit and leave room for future Claude-related targets without ambiguity.
