# Support Codex and Claude Code in v1

Harness v1 will support Codex and Claude Code as the first installation targets. These two targets define the initial adapter surface for translating the Éxito Plugin Manifest into agent-specific files and configuration.

## Consequences

The v1 scope is larger than a single-target CLI, but it validates the core architectural premise: one Éxito manifest can be adapted to multiple agent ecosystems. Other targets such as GitHub Copilot should wait until the manifest and adapter boundary prove stable.
