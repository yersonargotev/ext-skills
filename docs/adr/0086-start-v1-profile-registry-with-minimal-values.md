# Start v1 profile registry with minimal values

The v1 Catalog Profile Registry starts with exactly `role:frontend`, `role:fullstack`, `technology:react`, and `technology:nextjs`, because these values cover the initial discovery audience without carrying forward broader placeholder values like backend, infra, TypeScript, Python, or C#. `role:fullstack` is accepted as its own audience profile, not as a composed profile or shorthand for `frontend` plus `backend`; additional registry values require explicit later decisions.
